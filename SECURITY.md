# Política de Segurança

## Versões Suportadas

Atualmente, oferecemos suporte de segurança apenas para a versão mais recente do Koda Manager.


## Relatando uma Vulnerabilidade

Levamos a segurança da comunidade a sério. Se você descobrir uma vulnerabilidade de segurança neste projeto ;

1. Envie um e-mail ou DM no Discord para o desenvolvedor principal.
2. Descreva a vulnerabilidade com passos para reproduzir.
3. Aguarde a correção antes de divulgar publicamente.

Agradecemos sua colaboração para manter o ecossistema  seguro.
